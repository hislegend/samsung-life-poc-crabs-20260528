7 STRATEGIC PROPOSALS

PoC 성공을 위한
크랩스의 7가지 구체적 제안

단순한 제작 자동화가 아니라, 삼성생명 AI 서치 플랫폼 자체의 체질을 바꾸는 단계별 솔루션입니다.

📊
PROPOSAL 01

다양한 인풋 대응
모드별 커스터마이징

입력되는 정보의 성격에 따라 분석 보드를 완전히 달리하여

약관·교육자료·뉴스·상점 정보 등 각 모드별 최적 분석 로직을 N개 분리 제공합니다.

  • 약관 모드: 표 안의 표 구조까지 의미 단위 추출
  • 뉴스 분석 모드: 인물·사건·트렌드 3축 분석
  • 교육자료 모드: 핵심 화법·니즈 환기 키워드 자동 추출
  • 키워드 단위가 아닌 문맥 단위 추출 기술 적용
🎬
PROPOSAL 02

자막·음성·영상
3중 오버레이 퀄리티

최종 발표·시연 환경에서 가장 민감한 립싱크·음성·자막 정합성 이슈를 원천 차단. 

GS에서 이미 적용한 검증된 기술입니다.

  • 음성 어색함 발생 시 자동 오버레이 처리
  • 자막 깨짐·오타 깨짐 현상 자동 검수
  • 립싱크·음성 자막·정합성 검수로 최종 시연 안정성 확보
  • 화면-음성 딜레이 이슈 대응 가이드 제공
🛡️
PROPOSAL 03

컴플라이언스 자동
검증 모듈 추가

"업계 최대" 같은 과장 표현, 금지 조항 위배 여부를 AI가 자체 판별해

보험 콘텐츠 신뢰도를 잃지 않으면서 재미를 살리는 균형점을 확보합니다.

  • 삼성생명 내부 컴플라이언스 규정 학습 적용
  • 걸린 항목과 근거 조항을 함께 제시
  • 인포그래픽 오류·디자인 패턴 깨짐도 자동 검수
  • 장기적으로 고객 배포 가능 콘텐츠 자동 판별
✂️
PROPOSAL 04

롱폼 → 숏폼
하이라이트 자동 추출

기존 상품 강의·EBS 특강 등의 긴 영상을 

수분 내에 하이라이트만 추출해 숏폼 형태로 바꿉니다. JTBC·KBO 등에 적용 중인

LTS(Long to Short) 기능입니다.

  • 영상 전체를 텍스트 인덱싱 후 핵심 장면 추출
  • 채팅으로 "미국 금리 부분 찾아줘" 식 장면 검색
  • 자체 비디오 에디터 + 릴스 컷 편집 자동화
  • 5분 영상도 3분 안에 검수형 컷 편집 가능
⚡
PROPOSAL 05

큐 기반 자동화
병렬 생성 시스템

"신경 안 쓰시고 일하다가 보시면 완성돼 있는" 구조. 

1분 영상 = Veo 8초 클립 × 8개 연장. 일일이 클릭 없이 큐가 차곡차곡 처리됩니다.

  • 영상 1개당 15~20분 생성 시간 자동 대기열 처리
  • 병렬 처리로 다수 콘텐츠 동시 생성 가능
  • 앞부분 뉘앙스 이어짐 보장 / 일괄 생성 선택 옵션
  • 단일 생성 도구 의존도 완화
🔐
PROPOSAL 06

망분리 환경 대응
유연한 납품 형태

삼성생명 내부 보안 환경을 그대로 존중. 외부 PC에서 작업 후 결과물만 들여오는 

현재 운영 흐름을 깨지 않습니다.

  • 기본: 인터넷 서비스 형태 제공
  • 고사양 요구 시: 전용 작업 환경 또는 별도 애플리케이션 제공
  • CDN 외부 스트리밍 서비스 옵션 가능
  • 화면 캡처가 어려운 자료는 인쇄물·사진 업로드도 지원
🤝
PROPOSAL 07

"같이 만들면서" 협업하는 운영 방식 — GS PoC 성공 공식 그대로

크랩스의 차별점은 솔루션을 던지고 끝나지 않는다는 것. 매주 정기 회의 + 품평회로 함께 만들어보며 메뉴·플로우·콘셉트를 같이 진화시킵니다. 

삼성생명 파트 내부에서 만들고 있는 다음 주 신규 콘텐츠 1편부터 즉시 동시 제작에 들어갑니다.

  • 다음 주 콘텐츠 5편 + 인물·시나리오·기존 프롬프트 이력 공유 → 양사 동시 제작 시작
  • "매뉴얼 만들 듯" 매뉴얼화 → 구축 전에 흐름과 프롬프트를 함께 설계
  • 품평회 방식 — "이거 위주로 뽑네" → 리버스 엔지니어링으로 회사 톤 학습
  • 아웃풋 틀 고정 금지 — 크랩스는 오픈 마인드로 제안, 삼성은 "선 넘으면" 피드백
  • 샘플 영상·기존 제작 프롬프트 → 학습 소스 확보 → 향후 확장 기반 마련
  • 인풋 결정 → 아웃풋 결정 → 중간 단계는 함께 맞춰가는 단계적 빌드
